Dream Color Catcher
Copyright (C) 2015 Yuesong SHEN

Description:

Dream Color Catcher, Catch the color of your life!

This software enables you to catch the color that impresses you from anywhere: your screen, your photo or even within your mind! Your dream color can be taken with just a few clicks! Enjoy!

This software is tested on Windows 7 x64, and might also work on other windows operation systems. Any tests, bug repports or suggestions are the most welcome! 

This is a simple program which implements one of my ideas. The language of the source code is VB6.
